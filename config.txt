Install allauth

*pip install django-allauth

In settings.py file add below configuration

AUTHENTICATION_BACKENDS = (
    'django.contrib.auth.backends.ModelBackend',
    'allauth.account.auth_backends.AuthenticationBackend',)

SOCIALACCOUNT_PROVIDERS = { 
    'google': {
        'SCOPE': [
            'profile',
            'email',
        ],
        'AUTH_PARAMS': { 
            'access_type':'online',
        }
    }
}

go to login.html & add social account

{% load socialaccount %}

we have added urls account in project urls.py
path('accounts/',include('allauth.urls'))

we have ran migrate
python manage.py migrate
